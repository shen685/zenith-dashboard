# Zenith Dashboard 📊

[![Next.js](https://img.shields.io/badge/Next.js-14.x-black.svg)](https://nextjs.org/)
[![Recharts](https://img.shields.io/badge/Recharts-2.x-22b5bf.svg)](https://recharts.org/)
[![Framer Motion](https://img.shields.io/badge/Framer_Motion-11.x-ff0055.svg)](https://www.framer.com/motion/)

**Zenith Dashboard** is a high-performance, AI-powered data visualization platform. It provides real-time insights with beautiful, interactive charts and a seamless user experience.

## 🌟 Key Features

- 📈 **Interactive Charts**: Powered by Recharts for smooth data exploration.
- 🤖 **AI Insights**: Integrated AI for automated data analysis and forecasting.
- 🌓 **Dark Mode**: Native support for dark and light themes.
- 📱 **Responsive**: Fully optimized for mobile, tablet, and desktop.
- ⚡ **Fast**: Built on Next.js 14 with App Router and Server Components.

## 🚀 Getting Started

```bash
git clone https://github.com/shen685/zenith-dashboard.git
cd zenith-dashboard
npm install
npm run dev
```

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Data Viz**: Recharts
- **Animation**: Framer Motion
- **Styling**: Tailwind CSS + Shadcn UI
- **State**: Zustand
